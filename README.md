# Investment-Thesis-Tracker
利用 AI 工具解讀財報，透過系統化練習，磨練對公司經營邏輯與整體產業趨勢的洞察力。

## 🧠 處理流程 (Workflow)
1. **AI 處理**：利用 AI 快速掃描財報、股東信與法說會逐字稿，提取關鍵數據與指標。
2. **邏輯建構**：分五大部分建構邏輯，以更多的Token數給出更完整的內容。
3. **進階觀點**：除了基礎觀點外，請AI給予深入觀點。
4. **驗證清單**：給予驗證清單，追蹤下一季度的財報有無達到目標。
