# 📈 NVIDIA (NVDA) 研究專區

此資料夾專門存放 NVIDIA (NVDA) 的財報拆解、資料中心市佔率、GPU 技術路徑及全方位 AI 生態系 (CUDA/Omniverse) 的觀察紀錄。

## 🎯 核心投資論點 (The Core Thesis)
> **「NVIDIA 不僅是晶片供應商，它是現代 AI 基礎設施的架構師，定義了從矽片、系統軟體到超大型資料中心運算的工業標準。」**

### 1. 護城河：CUDA 軟體平台與開發者生態
* **邏輯**：藉由超過十年的 CUDA 累積，NVIDIA 建立了一個軟硬體高度耦合的生態系。開發者對其軟體棧的黏性，使得競爭對手不僅要競爭硬體效能，更要克服軟體遷移的巨大成本。

### 2. 成長動能：主權 AI 與企業級推論需求
* **邏輯**：除超大規模雲端服務商 (Hyperscalers) 外，各國政府 (Sovereign AI) 與傳統企業對本地 AI 基礎設施的需求正進入爆發期，Blackwell 架構將帶動單價 (ASP) 與毛利進一步提升。

### 3. 戰略優勢：系統級銷售與 Spectrum-X 網路技術
* **邏輯**：NVIDIA 已從銷售單一 GPU 轉向銷售「整組機櫃 (GB200)」與網路交換機 (InfiniBand/Ethernet)，透過整合 NVLink 技術提供極低延遲的運算集群，這是單純晶片廠難以複製的系統級優勢。
